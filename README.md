# SQL Sales Analysis Project

## Objective
Analyze sales dataset using SQL queries.

## Tools Used
- MySQL Workbench
- SQL

## Key Analysis
- Top Customers
- Average Order Value
- Sales by Category
- Sales by Region
- Product Analysis

## Skills Used
- SELECT, WHERE
- GROUP BY, ORDER BY
- Aggregations (SUM, AVG)
- CASE Statements

## Dataset Columns
order_id, customer_name, category, product_name, total_price, region
